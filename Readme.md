## 📝 Resumo do Projeto
O **LucroImpresso** é um sistema web desenvolvido para solucionar a dificuldade de precificação no mercado de impressão 3D e modelagem de action figures. O sistema automatiza o cálculo do custo real de produção de uma peça, cruzando dados como o peso do material gasto, o tempo da máquina operando e o custo do consumo de energia elétrica. O objetivo é substituir o uso de planilhas complexas ou métodos de precificação baseados em "chutômetro", garantindo aos criadores a visualização exata dos seus custos operacionais e da sua margem de lucro real.

## 🛠️ Tecnologias Utilizadas
- **Linguagens:** HTML5, CSS3 e JavaScript (Vanilla).
- **Framework:** Bootstrap (utilizado para agilizar a criação da interface visual e garantir a responsividade em dispositivos móveis).
- **Banco de Dados / Backend:** Firebase (utilizando o *Firestore* para o armazenamento NoSQL dos orçamentos e materiais em nuvem, e o *Firebase Authentication* para o sistema de login seguro).
